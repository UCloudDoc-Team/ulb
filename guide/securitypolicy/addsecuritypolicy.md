# 创建安全策略

## 操作步骤

1、进入**负载均衡 ULB**页面，点击**安全策略管理**

2、点击**创建安全策略**，弹出创建安全策略的弹窗。

![](/images/创建安全策略.png)

3、需要填写以下信息

| 配置             | 说明                                                                    |
| ---------------- | ----------------------------------------------------------------------- |
| 名称             | 安全策略名称，必填项。                                                  |
| TLS协议最低版本  | TLS协议最低版本，必填项，支持可选的 TLS 版本为 TLSv1，TLSv1.1，TLSv1.2  |
| 加密算法套件     | 加密算法套件选择，必填项，支持的加密套件见[安全策略](securitypolicy.md) |
