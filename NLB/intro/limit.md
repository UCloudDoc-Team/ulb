# 使用限制

## NLB配额限制（本期暂不支持）

| **配额名称**                               | **配额** |
| ------------------------------------------ | -------- |
| 单实例可绑定EIP数                          | 30       |
| 单地域可创建实例数                         | 200      |
| 单实例可添加监控器数                       | 50       |
| 单实例可添加的转发规则数（不包括默认规则） | 100      |
| 单实例可添加服务节点数                     | 300      |
