# 使用限制

## NLB配额限制（本期暂不支持）

| **配额名称**                               | **配额** |
| ------------------------------------------ | -------- |
| 单实例可绑定EIP数                          | 30       |
| 单地域可创建实例数                         | 200      |
| 单实例可添加监控器数                       | 50       |
| 单实例可添加的转发规则数（不包括默认规则） | 100      |
| 单实例可添加服务节点数                     | 300      |



## NLB支持的地域

| 已支持地域   | 华北二，华北一，上海二，广州，香港，法兰克福                 |
| ------------ | ------------------------------------------------------------ |
| 暂未支持地域 | 福建，台北，新加坡，东京，首尔，曼谷，雅加达，胡志明，马尼拉，孟买，洛杉矶，华盛顿，圣保罗，莫斯科，伦敦，迪拜，拉各斯 |
