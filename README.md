
# 负载均衡 ULB

负载均衡ULB（UCloud Load Balancer）提供业务流量自动分发服务，通过将流量按需转发至后端多个云资源来提升业务系统的处理能力，并通过健康检查消除单点故障，提高业务系统可用性。即开即用，按需付费。

负载均衡ULB产品族包含：应用型负载均衡ALB、传统型负载均衡CLB。网络型负载均衡NLB即将上线。
